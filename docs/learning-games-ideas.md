# 교육용 학습 게임 아이디어

대상 연령: 4~7세
목적: 친구 키우기 게임과 자연스럽게 어우러지는 학습형 미니게임

---

## 🔢 숫자 / 수 개념

### 1. 뼈다귀 세기 ⭐⭐⭐⭐⭐
- 화면에 뼈다귀 N개(1~10) 흩뿌림
- "몇 개일까?" → 1~10 숫자 카드 중 정답 탭
- 정답 시 뼈다귀가 동물에게 날아가고 hunger +N
- **연령**: 4~5세
- **학습**: 1~10까지 수 세기

### 2. 같은 수 짝 맞추기
- 카드 12장: 점 그림(⚫⚫⚫)과 숫자(3) 매칭
- 사물(사과/별/하트)로도 변형 가능
- **연령**: 5~7세
- **학습**: 수와 양의 대응

### 3. 더하기
- 🍪 + 🍪 = ? (보이는 사물 합산)
- 답 1~10 중 탭
- 단계별 난이도
- **연령**: 5세 이상
- **학습**: 한 자리 덧셈

### 4. 큰 수 작은 수
- 두 숫자 비교 → "어느 게 더 커?"
- **연령**: 6~7세
- **학습**: 수 비교

### 5. 순서대로 누르기
- 1~5 풍선 화면 곳곳에 → 순서대로 터뜨리기
- 발자국 따라가기 패턴 변형
- **연령**: 4~6세
- **학습**: 수 순서

---

## 🇰🇷 한글

### 6. 첫소리 게임 ⭐⭐⭐⭐⭐
- 그림 4개 (사과/배/공/고양이)
- "ㄱ으로 시작하는 건?" → 정답 탭
- 자음 14개
- **연령**: 4~6세
- **학습**: 자음 인지 + 음운 인식

### 7. 글자 짝 맞추기
- "강아지" 글자와 🐶 그림 매칭
- 8~12개 짝
- **연령**: 5~7세
- **학습**: 글자 읽기

### 8. 한글 따라쓰기 ⭐⭐⭐⭐
- 큰 글자 위 점선 따라 손가락으로 그리기
- ㄱ, ㄴ, ㄷ ... 차례로
- 캔버스 사용
- **연령**: 5~7세
- **학습**: 글자 쓰기 / 운필력

### 9. 받침 찾기 (어려움)
- "사ㄴ" / "사ㅇ" / "사ㅁ" 중 "산" 만들기
- **연령**: 6~7세
- **학습**: 종성 인지
- ⚠️ 학습 부담 큼

### 10. 한글 단어 만들기
- 자음 ㄱ + 모음 ㅏ = "가"
- **연령**: 5세 이상
- **학습**: 음절 조합

## 🇺🇸 영어 (알파벳)

### 11. 대소문자 짝 맞추기 ⭐⭐⭐⭐
- A ↔ a 매칭
- 26쌍 중 6쌍씩 라운드
- **연령**: 5~7세
- **학습**: 대소문자 인지

### 12. 알파벳 순서
- A, B, ?, D, E → C 고르기
- ABC 송 활용 가능
- **연령**: 6~7세
- **학습**: 알파벳 순서

### 13. 첫글자 매칭 ⭐⭐⭐⭐⭐
- 🐶 그림과 D / B / C 중 D 탭
- "Dog!" 풍선/음성
- 동물/색깔/숫자 단어
- **연령**: 5~7세
- **학습**: 단어 첫소리 + 어휘

### 14. 색깔 단어 매칭
- 🔴 그림 → "Red" 글자 매칭
- 7~8개 색
- **연령**: 5~7세
- **학습**: 색깔 영어

### 15. 알파벳 노래 (인터랙티브)
- 알파벳이 차례로 떠오르고 누르면 음성/멜로디
- **연령**: 4~6세
- **학습**: 알파벳 노출

---

## 🌈 색깔 / 모양

### 16. 같은 색 찾기 ⭐⭐⭐⭐⭐
- 빨강 공 6개 + 파랑 공 1개 → 다른 거 찾기
- **연령**: 4~5세
- **학습**: 색깔 변별

### 17. 모양 맞추기
- ⭕ ⬛ 🔺 도형 슬롯에 떨어지는 도형 분류
- 단순한 테트리스
- **연령**: 4~6세
- **학습**: 도형 변별

## 🎯 통합 추천 — 우선순위

게임 톤과 잘 맞고 구현 부담 적은 순:

| 순위 | 게임 | 추천 이유 |
|---|---|---|
| 🥇 | **첫글자 매칭 (한/영)** | 그림+글자 매칭, 어휘 학습 직접, 모든 연령 |
| 🥈 | **뼈다귀 세기** | 친구 키우기 컨셉과 자연스럽게 어우러짐 |
| 🥉 | **같은 색 찾기** | 4세도 가능, 즉시 보상 |
| 4 | **대소문자 짝 맞추기** | 기존 짝 맞추기 코드 재활용 |
| 5 | **한글 따라쓰기** | 손가락 학습, 부모 만족도 |

## 구현 가이드라인

### 메뉴 구조
- 놀이 메뉴에 [🎓 학습] 탭 추가
- 일반 미니게임과 분리

### 일일 한도
- 학습 게임은 일일 한도 별도 또는 무제한
- "학습은 많이 할수록 좋다" 메시지

### 보상 체계
- 정답 N개 = 케어포인트 +M
- 일정 횟수 채우면 진화점수 보너스
- 부모 만족 요소 강조

### 난이도 자동 조절
- care 점수 기반 단계 풀림
- 처음: 1~5 / 색 3종 / 자음 5개
- 중간: 1~10 / 색 7종 / 자음 14개
- 고급: 더하기 / 단어 매칭 / 받침

### 음성 (옵션)
- Web Speech API는 한국어 품질 낮음
- 사전 녹음 mp3 권장 (가족 목소리 추천)
- 또는 의성어 풍선 + 효과음 보강

### 접근성
- 큰 글자 모드
- 색맹 친화 색상
- 사운드 토글

---

## 학습 모드 설정

보호자 모드 → "학습 강조" 토글
- 켜짐: 미션에 학습 게임 가중, 보상 더 큼
- 꺼짐: 일반 게임만

# 🐶 우리 강아지 (P0 MVP)

어린이(4~7세)용 다마고치 스타일 웹 게임.
바닐라 HTML/CSS/JS, 빌드 도구 없음. localStorage에 자동 저장.

> 강아지는 **절대 죽지 않아요.** 게이지가 0이 되어도 슬픈 표정만 짓고, 액션을 누르면 다시 회복합니다.

## 폴더 구조

```

├── style.css           파스텔 + 큰 둥근 버튼

│   │   └── _gen.py         (스프라이트 생성 스크립트, 게임 실행에는 불필요)

└── .gitignore
```

## 실행 방법

브라우저로 `index.html`을 직접 열어도 동작하지만, 모바일 LAN에서 같이 보려면 정적 서버로 띄우는 게 편해요.

```bash
cd /Users/niee/geminiProject/games/dogs
python3 -m http.server 11434 --bind 0.0.0.0
```

같은 와이파이의 폰/태블릿에서:

```
http://<맥의 LAN IP>:11434/
```

종료는 터미널에서 `Ctrl+C`. (현재 sprite-gen이 11434를 쓰고 있다면 먼저 종료해야 합니다.)

## 게임 규칙

- 4가지 게이지 (각 0~100): **🍖 배고픔 / 💖 행복 / 🫧 청결 / ⚡ 에너지**
- **30초마다 -5씩** 자연 감소
- 4가지 액션 버튼:
  | 버튼 | 효과 |
  | --- | --- |
  | 🍖 먹이 | 배고픔 +50, 행복 +5, 청결 -15 |
  | 🎾 놀기 | 행복 +35, 에너지 -15, 배고픔 -10, 청결 -10 |
  | 🛁 씻기 | 청결: 드래그 거리 누적 (+1/30px), 행복 -10, 에너지 -5 |
  | 💤 재우기 | 에너지 +60, 행복 +5, 배고픔 -15 |
- 강아지 표정은 게이지에 따라 자동으로 바뀝니다:
  - 평균이 높으면 **happy**
  - 에너지가 낮으면 **sleeping**
  - 게이지 중 하나라도 20 이하 → **sad**
  - 액션을 누르는 동안 잠깐 해당 표정으로 (먹이=eating 등)

## 저장 / 초기화

- 자동 저장: `localStorage`(`dogs.p0.state.v1`)에 게이지와 마지막 시각이 저장됩니다.
- 탭을 닫고 다시 열어도, 닫혀있던 시간만큼 게이지가 자연 감소한 상태로 이어집니다.
- 처음부터 다시 시작하고 싶으면 브라우저 콘솔에서:

  ```js
  __dogs.reset()
  ```

## 디자인 메모

- 폰트: Google Fonts **Jua / Gowun Dodum** (둥글고 친화적)
- 배경: 파스텔 베이지 + 하늘색 그라데이션
- 버튼: 최소 100×100, 둥근 모서리, `:active`에서 살짝 눌리는 효과 (모바일 터치 우선)
- 색감: 파스텔. 게이지가 낮아도 빨간 경고색 대신 부드러운 펄스 애니메이션만 사용
- 모션 민감 사용자 대응: `prefers-reduced-motion` 시 모든 애니메이션 비활성
